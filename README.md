# Logic-Gate-Simulator-MATLAB
Interactive Digital Logic Gate Simulator built using MATLAB App Designer with Boolean expression and truth table generation.


An interactive **Digital Logic Gate Simulator** developed using **MATLAB App Designer**. This application provides a simple and intuitive graphical user interface (GUI) to simulate the behavior of basic digital logic gates, display Boolean expressions, and automatically generate truth tables for educational purposes.

---

## Overview

Digital logic gates are the fundamental building blocks of digital electronic systems. This project helps students understand the operation of logic gates through real-time simulation without the need for physical hardware.

The simulator supports the seven fundamental logic gates and provides instant output verification along with the corresponding Boolean expression and truth table.

---

## Features

- Interactive GUI developed using MATLAB App Designer
- Supports seven basic logic gates:
  - AND
  - OR
  - NOT
  - NAND
  - NOR
  - XOR
  - XNOR
- Binary input selection
- Automatic logic output generation
- Boolean expression display
- Truth table generation
- Reset functionality
- Beginner-friendly interface

## How It Works

1. Launch the application.
2. Select a logic gate from the drop-down menu.
3. Choose the binary input values.
4. Click the **Simulate** button.
5. View:
   - Logic output
   - Boolean expression
   - Truth table
6. Click **Reset** to perform another simulation.

---

## Supported Logic Gates

| Logic Gate | Boolean Expression |
|------------|--------------------|
| AND | Y = A · B |
| OR | Y = A + B |
| NOT | Y = A' |
| NAND | Y = (A · B)' |
| NOR | Y = (A + B)' |
| XOR | Y = A ⊕ B |
| XNOR | Y = (A ⊕ B)' |
